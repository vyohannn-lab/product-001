
# 로또 번호 추첨기 블루프린트

## 개요

이 애플리케이션은 사용자에게 무작위 로또 번호를 생성하는 간단하고 매력적인 방법을 제공합니다. 현대적인 웹 기술을 사용하여 구축되어 시각적으로 매력적이고 사용자 친화적인 경험을 보장합니다.

## 스타일, 디자인 및 기능

*   **언어:** 한국어
*   **핵심 기능:** 버튼 클릭 시 1부터 45까지의 숫자 중 6개의 고유한 숫자를 생성합니다.
*   **디자인:**
    *   깨끗하고 현대적인 미학.
    *   생성된 숫자를 시각적으로 구별하기 위해 각각 다른 색상의 공으로 표시합니다.
    *   모바일 반응형 디자인.
    *   "번호 생성" 버튼에 대한 호버 및 클릭 효과.
*   **기술:**
    *   핵심 로직 및 UI를 위한 `lotto-generator`라는 웹 컴포넌트.
    *   컴포넌트 스타일링을 위한 Shadow DOM.
    *   전체적인 페이지 레이아웃 및 디자인을 위한 현대적인 CSS.

## 구현 계획

1.  **`index.html`:**
    *   언어를 한국어로 설정합니다.
    *   페이지 제목을 "로또 번호 추첨기"로 변경합니다.
    *   `lotto-generator` 웹 컴포넌트를 포함할 본문을 설정합니다.
2.  **`main.js`:**
    *   `lotto-generator` 커스텀 엘리먼트를 정의합니다.
        *   Shadow DOM을 사용하여 컴포넌트의 HTML 구조와 스타일을 캡슐화합니다.
        *   6개의 고유한 난수를 생성하는 로직을 구현합니다.
        *   각 숫자를 다른 색상의 공으로 표시하는 기능을 추가합니다.
        *   "번호 생성" 버튼에 대한 이벤트 리스너를 추가합니다.
3.  **`style.css`:**
    *   페이지의 전반적인 스타일(배경, 글꼴 등)을 정의합니다.
    *   `lotto-generator` 컴포넌트가 중앙에 오도록 레이아웃을 조정합니다.
